Detailed Description

Simplify INT_SLESS applied to 0 or -1.

Forms include:

  • 0 s< V * -1 => V s< 0
  • V * -1 s< 0 => 0 s< V
  • -1 s< SUB(V,#hi) => -1 s< V
  • SUB(V,#hi) s< 0 => V s< 0
  • -1 s< ~V => V s< 0
  • ~V s< 0 => -1 s< V
  • (V & 0xf000) s< 0 => V s< 0
  • -1 s< (V & 0xf000) => -1 s< V -CONCAT(V,W) s< 0 => V s< 0 --1 s< CONCAT(V,W) => -1 s> V`

There is a second set of forms where one side of the comparison is built out of a high and low piece, where the high piece determines the sign bit:

  • -1 s< (hi + lo) => -1 s< hi
  • (hi + lo) s< 0 => hi s< 0
